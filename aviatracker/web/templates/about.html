About the project

MyFly is a service, providing real-time access to air traffic data. It uses The OpenSky Network API for retrieving information about aircraft location.

Data from the OpenSky Network API

As soon as a message of an airplane arrives at servers, OpenSky Network creates a record for the aircraft – the so-called state vector. This state vector represents a summary of all tracking information at a certain point in time, including aircraft identity (ICAO address + call sign), time information (Unix timestamps), and spatial information (position, velocity, heading, …).

All vectors represent the state of a vehicle with the interval [time−1, time]. If OpenSky doesn't receive any position or velocity update for 15 seconds, the state vector is considered obsolete and the API won’t return any state vector for the respective airplane.
